Вы можете запускать код сниппета автоматически

Будьте осторожны при использовании настроек автозапуска! Тестируйте свой код перед сохранением, иначе придется чистить базу!
Небольшая справка:

Вы можете использовать своем сниппете JS,PHP и CSS код

  • CSS размещайте так:
    <style> #main{ display:block; } </style>
  • PHP размещайте так:
    <?php echo "123"; ?>

    Вы можете работать с содержимым сниппета используя переменную $content
    ПРИМЕР: вставьте в текст материала такой шорткод: [rsnippet name="somename"]1+2=[/rsnippet]
    И вставьте это в код сниппета

    <?php $content .= "3"; echo $content; ?>

  • JS размещайте так:

    <script type="text/javascript"> alert("Hello World!"); </script>

  • Для JQuery
    <script type="text/javascript"> jQuery(function($) {
		$(document).ready(function(){
			// write your code here
		});
	});
</script>
